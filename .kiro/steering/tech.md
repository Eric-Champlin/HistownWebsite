# Technology Stack

## Core Technologies
- **Frontend Framework**: React with TypeScript
- **Styling**: Tailwind CSS
- **Build System**: Modern React tooling (likely Vite or Create React App)

## Development Approach
- Component-based architecture with reusable UI components
- TypeScript for type safety and better developer experience
- Tailwind for utility-first CSS styling
- Responsive-first design approach
- Focus on accessibility and modern web standards

## Design Principles
- Clean, modern layouts with soft gradients and movement
- Mobile-responsive design
- Faith-centered but contemporary aesthetic
- Clear navigation and user-friendly interfaces
- Performance-optimized for fast loading

## Common Commands
*Note: Specific build commands will be determined based on the chosen build system*
- Development server: `npm run dev` or `yarn dev`
- Build for production: `npm run build` or `yarn build`
- Type checking: `npm run type-check` or `yarn type-check`
- Linting: `npm run lint` or `yarn lint`
